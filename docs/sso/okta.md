# Okta OIDC groups to Cordum roles

The canonical operator guide is published on the Cordum docs site at <https://docs.cordum.io> (authored in `Cordum-site/docs-site`, not this repo).

Use it to configure Okta's standard `groups` claim and Cordum's `CORDUM_OIDC_GROUPS_CLAIM` / `CORDUM_OIDC_GROUP_ROLE_MAPPING` settings.
